# Tutorial de instalação em português

Passo 1: Descompacte o arquivo whmcs-gateway-preferences.zip

Passo 2: Acesse a pasta whmcs-gateway-preferences, resultante da descompactação, dentro dela terá uma pasta "modules".

Passo 3: Acesse a raiz de instalação do seu WHMCS

Passo 4: Arraste e solte a pasta "modules" da pasta descompactada para a raiz do seu WHMCS.

Passo 5: Acesse o seu WHMCS e ative o módulo "Preferências de Gateway"

# Installation tutorial in english

Step 1: Unzip the whmcs-gateway-preferences.zip file

Step 2: access the whmcs-gateway-preferences folder, resulting from the unziping, inside it will have a "modules" folder.

Step 3: Access your WHMCS installation root

Step 4: Drag and drop the "modules" folder from the unzipped folder to the root of your WHMCS.

Step 5: Go to your WHMCS admin and activate the module "Gateway Preferences".
